## Challenges Faced

- Fixing incorrect certificate links
- Understanding bucket policies
- Configuring CloudFront
- Learning the AWS Console
- Understanding website endpoints vs bucket endpoints

---

## Key Takeaways

- Always use IAM instead of the root account.
- CloudFront improves performance and enables HTTPS.
- Small filename mistakes can break deployments.
- Documentation is as important as deployment.
- Git is essential for tracking project history.
